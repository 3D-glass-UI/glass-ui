Details

How it differs

from regular glassmorphism

This library came about because the glassmorphism CSS I saw online consisted of lackluster translucency with a bit of blur and a thin white outline. So I experimented with a bunch of freshly supported CSS to develop a more refined and three-dimensional glass look.

I also tried to get ChatGPT 4 to write this glass CSS for me...

The above output is fairly representative of the glassmorphism tutorials currently available online. Not a noise texture or glistening specular highlight in sight.

Key characteristics

of next-gen glassmorphism

  • No white outlines!
  • Specular highlights
  • Fine-grained noise textures
  • Large blur radius
  • Bevels that fade smoothly into non-existence

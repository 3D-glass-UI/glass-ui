What is Glassmorphism 3.0?

Glassmorphism 3.0 employs a combination of backdrop-filters, a complex box-shadow layering technique, and a noise texture to create a to create a glass effect so realistic, you'll be trying to wipe your fingerprints off the screen.

  • Box-shadow layering instead of border
  • Big blur radius
  • Brightness & saturation filters
  • Subtle noise texture
  • Mask-composite gradient to create specular highlights

  /* Add your own noise texture here! */

  background-size: 100px;
  background-repeat: repeat;
}

Note

If you apply vibrant glass to an element and it looks wrong or seethrough, then a new stacking has been context is created by another class or in a papent element. You can either fix the stacking context or just use frosted glass instead.

::after

.frosted-glass::after {
  position: absolute;
  content: "";
  inset: 0;
  border-radius: inherit;
  z-index: -2;
  background-color: rgba(250, 254, 255, 0.12);
  mix-blend-mode: multiply;
}

Beveled glass tutorial

Vibrant glass uses the mix-blend-mode property to add dynamic depth of color to the glass material.

View vibrant glass tutorial

Hide vibrant glass tutorial

Abandon all hope, ye who's wondered thus far.' Be warned that this technique is quite finicky to implement and will stop working if applied to an element that has its own stacking context. The complexity arises from the interplay of multiple CSS properties, that each automatically create a new stacking context. Furthermore, mix-blend-mode needs to be set up such that it has access to the page’s backdrop to bend with. Finally, it is important to keep text above the glass so that its legibility is not affected.

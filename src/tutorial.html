Elevation box-shadow

The elevation box-shadow plays a pivotal role in producing the illusion of a three-dimentional pane of glass.

More info

Less info

Do you really need it?

I would say yes, unless you are going for a Hawaiian Lāhainā Noon vibe.

How to use it

The shadow can either be added to the main element of a component class or via HTML. I chose not to include it in the .glass-base class because its size value is too variable.

if: No interactions or animations

then: add it to the component class, and consider giving it a couple breakpoints.

or: if it’s not a reusable component and you don’t want it to adapt to the size of the viewport, then just drop it into the HTML

if: CSS interactions & animations

then: add it to the component class and consider giving it a couple breakpoints.

e.g. .btn-glass-light includes a box-shadow that changes size on :hover

if: JS interactions & animations

then: add it as .sha[t-shirt size] class to the HTML so that it can be easily swapped via JS

Naturally, the first question is:

Can chatGPT write this CSS?

Answer: nope this ain’t it.

The above output is fairly representative of the glassmorphism tutorials currently available online. Not a noise texture or glistening specular highlight in sight. The tiny blur radius compromises legibility, while the bright white outline falls lightyears short of the dimensionality it is meant to depict.

But fear not! We can do better. Keep reading, and I'll show you how.

chatGPT 4's glass CSS

.glass CSS

.glass-base {
  position: relative;
  -webkit-backdrop-filter: blur(var(--blur)) brightness(0.95) saturate(1.3);
  backdrop-filter: blur(var(--blur)) brightness(0.95) saturate(1.3);
}

.glass-base::after {
  content: "";
  position: absolute;
  inset: 0;
  pointer-events: none;
  border-radius: inherit;
  z-index: -1;

  box-shadow: 
    inset 1px 1px 0.2px rgba(250, 254, 255, 0.2),
    inset 2px 2px 1px -2.4px rgba(250, 254, 255, 0.8), 
    inset -2px -2px 0.4px -2.4px rgba(250, 254, 255, 0.1),
    inset 0 -0.5px 1px -1px rgba(250, 254, 255, 0.5);

  background-size: 100px;
  background-repeat: repeat;

/*** Add your own noise textures here! ***/
  background-image:
    url(/assets/images/noise-texture-500x500.png),
    url(/assets/textures/noise-texture-black-100den-10op-500x500.png);
}
